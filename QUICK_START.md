## 🎮 지금 바로 테스트하기

### 1. Godot 실행
1. 시작 메뉴에서 "Godot" 검색 후 실행
2. 또는 설치된 폴더에서 `Godot_v4.5.1-stable_win64.exe` 실행

### 2. 프로젝트 열기
1. Godot 실행 후 **"Import"** 버튼 클릭
2. 이 프로젝트 폴더 선택: `C:\Users\LENOVO\Desktop\Moon\Development\ddongstory`
3. `project.godot` 파일이 있는 폴더 선택
4. **"Import & Edit"** 클릭

### 3. 게임 실행
1. 왼쪽 파일 탐색기에서 `scenes/Main.tscn` 더블클릭
2. 상단의 **▶ (Play)** 버튼 클릭
3. 또는 `F5` 키 누르기

### 4. 조작 테스트
- **← → (방향키)** 또는 **A / D**: 좌우 이동
- **스페이스바** 또는 **↑**: 점프
- **ESC**: 게임 종료

## ⚠️ 문제 해결

### Godot이 실행되지 않을 때
- Windows Defender가 차단할 수 있음
- "추가 정보" → "실행" 클릭

### 프로젝트를 열 수 없을 때
- `project.godot` 파일이 있는지 확인
- Godot 버전이 4.3 이상인지 확인

### 플레이어가 보이지 않을 때
- `scenes/Main.tscn`이 열려있는지 확인
- Scene 탭에서 Player 노드가 있는지 확인

### 충돌이 안 될 때
- CollisionShape2D에 shape가 설정되어 있는지 확인
- Physics Layers 설정 확인
