# 기술 스택 선택 가이드

## 🎮 게임 엔진 옵션

### 1. **Unity** (C#)
**장점:**
- 강력한 2D/3D 지원
- 풍부한 에셋 스토어
- 널리 사용됨 (커뮤니티 크고 자료 많음)
- 멀티플랫폼 지원

**단점:**
- 상대적으로 무거움
- 학습 곡선이 있음

**추천도:** ⭐⭐⭐⭐

---

### 2. **Godot** (GDScript / C#)
**장점:**
- 가볍고 빠름
- 오픈소스 (무료)
- 2D 게임에 최적화
- 직관적인 에디터

**단점:**
- Unity보다 커뮤니티 작음
- 에셋이 상대적으로 적음

**추천도:** ⭐⭐⭐⭐⭐ (2D 게임에 매우 적합)

---

### 3. **Phaser** (JavaScript/TypeScript)
**장점:**
- 웹 기반 (브라우저에서 실행)
- HTML5 게임에 적합
- JavaScript/TypeScript 사용

**단점:**
- 웹 게임에 특화 (데스크톱 앱화는 추가 작업 필요)
- MMORPG 구현이 복잡할 수 있음

**추천도:** ⭐⭐⭐

---

### 4. **Pygame** (Python)
**장점:**
- Python으로 쉽게 시작
- 간단한 2D 게임에 적합

**단점:**
- MMORPG에는 부적합
- 성능 제한

**추천도:** ⭐⭐ (프로토타입용)

---
