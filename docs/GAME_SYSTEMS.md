# 게임 시스템 설계

## 🎯 핵심 시스템

### 1. 캐릭터 시스템

#### 기본 속성
- **레벨 (Level)**: 1 ~ 200
- **경험치 (EXP)**: 레벨업에 필요
- **체력 (HP)**: 생명력
- **마나 (MP)**: 스킬 사용에 필요
- **공격력 (ATK)**: 물리 공격력
- **마법력 (MATK)**: 마법 공격력
- **방어력 (DEF)**: 물리 방어력
- **마법 방어력 (MDEF)**: 마법 방어력
- **이동 속도 (SPD)**: 이동 속도
- **점프력 (JUMP)**: 점프 높이

#### 직업별 특성
- **똥냥이**: 높은 HP, DEF, 낮은 SPD
- **마법냥이**: 높은 MATK, MP, 낮은 HP, DEF
- **궁수냥이**: 높은 ATK, SPD, 중간 HP
- **도적냥이**: 높은 SPD, 크리티컬, 중간 HP
- **힐러냥이**: 높은 MP, 힐량, 중간 HP
- **소환냥이**: 높은 MP, 소환수 능력, 낮은 HP
- **변신냥이**: 균형잡힌 스탯, 변신 능력

### 2. 스킬 시스템

#### 스킬 구조
```javascript
{
  id: "skill_001",
  name: "똥 검 휘두르기",
  job: "똥냥이",
  type: "active", // active, passive, buff
  element: "physical", // physical, fire, ice, none
  cooldown: 0, // 쿨타임 (초)
  mpCost: 0, // 마나 소모
  damage: 100, // 기본 데미지
  range: 1, // 사거리
  description: "근접 공격 스킬"
}
```

#### 스킬 레벨 시스템
- 스킬 포인트로 스킬 레벨업
- 레벨이 올라갈수록 데미지/효과 증가
- 최대 스킬 레벨: 20

---

### 3. 전투 시스템

#### 데미지 계산
```
최종 데미지 = (기본 데미지 × 스킬 배율) - 방어력
크리티컬 = 최종 데미지 × 1.5
```

#### 속성 상성
- 불 > 얼음
- 얼음 > 불 (상대적으로 약함)
- 무속성 = 중립

---

### 4. 아이템 시스템

#### 아이템 타입
- **무기**: 공격력 증가
- **방어구**: 방어력 증가
- **악세서리**: 특수 능력
- **소비품**: 포션, 버프 아이템

#### 아이템 등급
- 일반 (흰색)
- 고급 (파란색)
- 희귀 (보라색)
- 전설 (주황색)

---

### 5. 맵 시스템

#### 맵 구조
- **타일 기반**: 2D 타일맵
- **레이어**: 배경, 지형, 오브젝트, 캐릭터
- **충돌 감지**: 타일 기반 충돌

#### 맵 타입
- **필드**: 일반 사냥터
- **던전**: 특수 던전
- **마을**: 안전 지역 (전투 불가)

---

### 6. 전직 시스템

#### 전직 단계
1. **1차 전직**: 레벨 10
2. **2차 전직**: 레벨 30
3. **3차 전직**: 레벨 60
4. **4차 전직**: 레벨 100

#### 전직 보상
- 새로운 스킬 해금
- 스탯 증가
- 특수 능력 획득
