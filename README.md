# 똥스토리 (Ddong Story) 🐱💩

고양이들이 주인공인 2D 사이드스크롤 MMORPG 게임

## 📋 프로젝트 개요

옛날 메이플랜드의 향수를 느낄 수 있으면서도 독창적인 직업 시스템과 스킬을 제공하는 고양이 RPG 게임입니다.

## 🎮 게임 컨셉

자세한 내용은 [GAME_CONCEPT.md](./GAME_CONCEPT.md)를 참고하세요.

## 🚀 시작하기

### 기술 스택
- **게임 엔진**: Godot 4.3+
- **언어**: GDScript
- **아트**: 2D 픽셀 아트

### 프로젝트 구조
```
ddongstory/
├── docs/              # 문서
├── assets/            # 게임 리소스
│   ├── sprites/       # 스프라이트
│   ├── sounds/        # 사운드
│   └── maps/          # 맵 데이터
├── src/               # 소스 코드
└── README.md
```

## ⚠️ 보완 필요 사항

게임 개발자가 실제로 느끼는 개선 포인트들:

1. **스탯 성장 공식** - 레벨업 시 스탯 증가 공식 정의 필요
2. **스킬 배율 구조 세분화** - 다타격, 상태이상, MP 소모 공식 등
3. **몬스터 시스템** - 몬스터 스펙, 경험치, 드랍 테이블 필요
4. **네트워크 구조** - 싱글 완성 후 멀티플레이어 추가 예정

자세한 내용은 [docs/IMPROVEMENTS.md](./docs/IMPROVEMENTS.md) 참고
